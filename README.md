A challenge dungeon built for streamer **Pony.gamer**! 🎮🐴

Fight your way through **10 levels** of increasingly tough challenges. Do you have what it takes to conquer the dungeon and claim the ultimate prize?

## 🎯 What Is This?

M1778-Dungen is a 10-level challenge dungeon designed to test your skills live on stream. Each level gets harder — and the rewards get bigger!

## 🚀 How to Run Locally

```bash
npm install
npm start
```

Then visit [http://localhost:3000](http://localhost:3000) in your browser. That's it!

## 🌐 How to Deploy

1. Push your code to GitHub
2. Go to your repo **Settings** → **Pages**
3. Enable GitHub Pages (the included workflow will handle the rest automatically on every push to `main`)
